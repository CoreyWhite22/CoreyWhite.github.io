In this exercise there are two buttons. The first button is an "Add PacMan" button, which adds stationary PacMen to your screen. The second button is a "Start Game" button, which makes each PacMan start moving at a random speed and bounce off any walls that they collide with.

In this exercise there is a set of eyes in the middle of the screen. Wherever we move our mouse the eyes follow us there.

In this exercise we use the Google Maps API to display the live location of buses on a map. We gathered real-time data of the Boston MBTA buses and added it to a Google Map. The bus location markers are updated every 15 seconds. The marker changes color to either blue or red depending on which direction it's going.
